# Unified RAG

A multimodal RAG app powered by **Gemini Embedding 2** and **Gemini 2.5 Flash**.

Upload images, PDFs, or ask questions — all in one unified embedding space.

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```

Requires a Google API key from [Google AI Studio](https://aistudio.google.com/app/apikey).
